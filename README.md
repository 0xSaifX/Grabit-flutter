# Grabit Marketing App

## Overview
This document outlines the structure and features of the Grabit marketing app, which includes various sections and functionalities to enhance user experience.

## Features
- **Header**: Contains the app name and logo, a search input, and icons for account creation, wishlist, and cart.
- **Hero Section**: An image with a button and text overlay.
- **Category Row**: Displays icons for fruits, vegetables, bakery items, snacks, drinks, and milk, with names and remaining amounts.
- **Product Cards**: Each card displays product image, description, name, rating, price, and a banner indicating if the product is new or on sale.
- **New Arrivals Section**: A section dedicated to new products with navigation buttons for stock names.
- **Customer Support Section**: Icons for guarantee items, payment options, and return policy.
- **Footer**: Contains categories, company information, and contact details with a dark theme.

## Pages
The app will include the following pages accessible via a drawer:
1. Grocery Page
2. Fashion Page
3. Jewelleries Page
4. Vegetables Page
5. Order Page
6. Cart Page

## Design
The app will have a modern and user-friendly design, focusing on ease of navigation and accessibility.

## Technologies
- Flutter
- Dart

## Conclusion
The Grabit app aims to provide a seamless shopping experience with a focus on marketing various products effectively.
